# ResumeAI - AI Resume Screening Platform

## Overview
ResumeAI is a full-stack AI-powered resume screening application that helps HR teams automatically extract skills, analyze resumes against job descriptions, and score candidates.

## Tech Stack
- **Frontend**: React + Vite + Tailwind CSS + ShadCN UI
- **Backend**: Node.js + Express + MongoDB
- **AI Service**: Python Flask with scikit-learn
- **Authentication**: Firebase Auth
- **File Parsing**: pdf-parse, mammoth

## Setup Instructions

### Prerequisites
- Node.js 18+
- Python 3.9+
- MongoDB Atlas account
- Firebase project

### Environment Variables
Create `.env` files in each directory.
